Estruturas condicionais

Estruturas condicionais, ou de seleção/decisão, permitem que, a partir da avaliação de uma expressão que gera um valor lógico, ou seja, um valor verdadeiro ou falso, o fluxo de execução do programa seja alterado. Elas têm papel fundamental na construção de algoritmos e, por consequência, na escrita de programas.

Estrutura condicional If(Se) e IF...Else (Se...Ou)

A forma mais simples de usarmos a estrutura condicional if é utilizá-la para executar algum trecho de código apenas se a condição que ela verifica (teste) é verdadeira. No entanto, é possível também é associar também a ela um trecho de código que será executado caso sua condição seja avaliada como falsa. Ou seja, temos um bloco de código que executa caso a condição seja verdadeira e um caso ela seja falsa.

  • Exemplo 2:
    • if ( maiorDeIdade ) { printf( "Voce eh maior de idade!" ); } else { printf( "Voce nao eh maior de idade!" ); }
